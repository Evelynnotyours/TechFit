Computer Science (B.C.Sc.) ▶️

Computer Science ဟာ အလျင်မြန်ဆုံး အခက်အခဲဖြေရှင်းမှုနည်းလမ်းတွေ၊ အယ်လဂိုရီသမ်များ၊ ဒေတာဖွဲ့စည်းပုံများ၊ Software Engineering၊ Database Systems၊ Artificial Intelligence၊ Parallel & Distributed Computing စတဲ့ software-အခြေခံ ဘာသာရပ်တွေကို အဓိကထား လေ့လာတဲ့ လမ်းကြောင်းပါ။

Career Path

1. Software Engineer / Developer
2. Web / Mobile App Developer
3. Data Scientist / Data Analyst
4. AI / Machine Learning Engineer
5. Database Administrator
6. Research & Teaching Assistant

Computer Technology (B.C.Tech.) ▶️

Computer Technology ဟာ Computer Architecture, Operating Systems, Networks, Digital Signal Processing, Embedded Systems စတဲ့ hardware/system-ပိုင်းကို အခြေခံပြီး စနစ်တကျ လေ့လာစေတဲ့ လမ်းကြောင်းပါ။ Network Security လည်း CT ထဲမှာပါဝင်ပြီး cyber security အခြေခံတွေကိုတွေ့ရပါမယ်။

Career Path

1. Network / System Engineer
2. Cyber Security Analyst (Network Security)
3. System Administrator / DevOps Associate
4. Embedded / IoT Developer
5. Cloud Infrastructure Technician
6. IT Support / Infrastructure Engineer
